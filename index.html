💡 Ser / Estar / Haber：三个高频“存在感”动词

ser 是一个动词原形（Infinitivo）。 它相当于英语里的 to be，真正造句时通常要按主语变位。

1. Ser 就等于英语里的 "to be"
英语里不会说 "I to be a student"，而是说 "I am a student"。西语同理。
• ser 是原形 (to be)
• soy, eres, es, somos, sois, son 是它的现在时变位
2. 原形什么时候出场？—— 一个简单谓语通常只变一个核心动词
当另一个动词已经变位，后面的动词经常保持原形。
• 单动词：El acuerdo es válido. (该协议是有效的。)
• 双动词连用：El acuerdo debe ser válido. (该协议必须是有效的。)
3. Estar 是什么？—— “处在某种状态/位置”
estar 也是动词原形，也常翻成“是/在”，但它主要表达状态、位置、临时情况。
• El contrato está listo. = 合同准备好了。强调当前状态。
• El documento está en la mesa. = 文件在桌上。强调位置。
• “它”也按第三人称单数走：Está aquí. = 它在这里。
4. Haber 是什么？—— 现代常用核心是 hay
haber 本身是动词原形。现在时里你最常用到的是非人称形式 hay，表示“有/存在”。
它不是按 yo/tú/él 去变的普通人称动词；不管后面是一个东西还是多个东西，都用 hay。

💡 Haber / Hay：表示“有/存在”，不是 estar 的普通变位

hay 来自动词 haber 的非人称用法，意思是“有、存在”。它不跟着东西的单复数或阴阳性变化。

• Hay un documento en la mesa. = 桌上有一份文件。
• Hay dos documentos en la mesa. = 桌上有两份文件。
两句都用 hay，不会因为 documento/documentos 改成别的形式。
Hay vs. Estar
• Hay un cliente en la oficina. = 办公室里有一位客户。强调“存在”。
• El cliente está en la oficina. = 那位客户在办公室。强调“已知的人在哪里”。
最实用的判断法
• 第一次提到“有某物/某人”：多用 hay。
• 已经知道这个东西/人，只说它在哪里或状态如何：多用 estar。
• 说身份、本质、国籍、职业、材料等稳定属性：多用 ser。

💡 名词性数：规则推测 + 常见例外

1. 阴阳性规律 (只能作为推测)：
• 多数 -o 结尾为阳性：documento。
• 多数 -a 结尾为阴性：firma。
• -ción, -dad, -tad, -tud 多为阴性；-ma 希腊来源词常为阳性，如 el problema。

2. 单复数变形逻辑：
• 元音结尾：加 -s，如 socio → socios。
• 辅音结尾：加 -es，如 papel → papeles。
• z 结尾：变 ces，如 luz → luces。
• 重音可能增减：resumen → resúmenes，resolución → resoluciones。

💡 规则动词现在时生成器：砍掉尾巴，替换分身

1. 三种原形长相：西语动词原形主要以 -ar, -er, -ir 结尾。

2. 规则动词原则：现在时要砍掉两个字母的原形尾巴，然后按人称贴上对应后缀。这个生成器会自动识别常见不规则动词；没收录的词仍可能需要查不规则表。

造句的潜规则：语序与省略

🧩 1. 修饰顺序：多数描述性形容词后置

西班牙语里，描述性质、类别、国籍、技术属性的形容词多数放在名词后面。不要把这个理解成绝对规则；数量、所属、某些常用评价词或强调词常常放在前面。

✅ 常见后置：
• 国际协议 → el acuerdo internacional
• 有效合同 → el contrato válido
• 技术规范 → el estándar técnico
✅ 常见前置：
• 所属：mi contrato, tu cliente, nuestro socio
• 常用短形容词：un buen día, un gran problema
• 时间/顺序：la próxima semana, el primer documento

✂️ 2. 主语省略：什么时候可以不说主语？

西语动词变位已经包含了很多人称信息，所以 yo, tú, nosotros 等代词常常省略。

💡 为什么？ 因为动词尾巴已经说明是谁在做动作。“一听尾巴，就知道是谁”，不需要每句都补代词。

📝 商务实战举例：
• (我) 审查协议 → Yo Reviso el acuerdo. (reviso 指向“我”)
• (我们) 和客户签署合同 → Nosotros Firmamos el contrato con el cliente. (firmamos 指向“我们”)

注：需要强调“就是我/就是我们”，或者第三人称可能歧义时，可以把主语代词补上。

🚫 3. 否定句语序：no 放在变位动词前

西语里最常见的否定词是 no。它通常直接放在已经变位的动词前面，位置比中文更固定。

基本公式：
主语（可省略） + no + 变位动词 + 其他内容
• No firmo el contrato. = 我不签合同。
• El documento no está listo. = 文件还没准备好。
• No hay problemas. = 没有问题。
双动词时：
no 放在第一个变位动词前，后面的动词保持原形。
• No quiero firmar. = 我不想签。
• No podemos revisar el acuerdo hoy. = 我们今天不能审查协议。

❓ 4. 疑问句语序：标点、语调、疑问词

西语直接疑问句要用一前一后两个问号：¿ ... ?。最常见有三种问法。

1. 是/否问题：陈述句加问号，靠语调提问
• Firmamos el contrato. = 我们签合同。
• ¿Firmamos el contrato? = 我们签合同吗？
2. 主语后置：动词先出场，更像正式书面问法
• ¿Está listo el documento? = 文件准备好了吗？
• ¿Tiene usted el contrato? = 您有合同吗？
3. 疑问词问题：疑问词放句首，后面接变位动词
• ¿Dónde está el documento? = 文件在哪里？
• ¿Cuándo firmamos? = 我们什么时候签？
• ¿Por qué no está listo? = 为什么还没准备好？

否定疑问句里，no 仍然放在变位动词前：¿No está listo? = 还没准备好吗？
