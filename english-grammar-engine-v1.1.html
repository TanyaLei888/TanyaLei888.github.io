英语先学句子骨架：谁 + 做什么 + 对谁/什么 + 在哪里/什么时候

核心骨架：主语 + 谓语 + 宾语

英语句子非常依赖位置。先抓住 who does what，再补时间、地点、方式、原因。

I sent the file to the client yesterday.
我昨天把文件发给了客户。
I = 谁；sent = 做了什么；the file = 做的对象；to the client = 给谁；yesterday = 什么时候。

英语修饰顺序

形容词多数放名词前：an important document, a technical issue
地点和时间通常放句尾：We met the client in Shanghai yesterday.
频率副词常放实义动词前、be 后：I often check emails. / She is always busy.

名词 / 冠词 / 代词：先解决“一个、这个、那些、谁的”

类别单数可数复数可数不可数用法感觉

Be / Have / There be：中文都可能像“是/有”，英语分工很清楚

Be：身份、状态、位置。She is a manager. / The file is ready.
Have：某人或某物拥有。I have a question. / She has a meeting.
There be：某处存在某人/某物。There is a file. / There are two files.

英语时态系统：时间 + 状态，而不是只背中文翻译

先分两层：什么时候 + 动作长什么样

时间：现在、过去、将来。状态：一般、进行、完成、完成进行。组合起来就是英语主要时态。

时态公式核心感觉例句
一般现在时do / does习惯、事实、规律、日程I work every day. / She works here.
现在进行时am/is/are + doing此刻正在发生；近期安排We are reviewing the file.
一般过去时did / was / were过去某个时间发生并结束They signed the contract yesterday.
过去进行时was/were + doing过去某一刻正在发生I was working at 9 p.m.
一般将来时will + do预测、决定、承诺We will send the file tomorrow.
将来进行时will be + doing将来某一刻正在进行I will be meeting the client at 3.
现在完成时have/has + done过去动作对现在有影响；到现在为止She has finished the report.
过去完成时had + done过去的过去They had left before I arrived.
将来完成时will have + done到将来某时已经完成We will have completed it by Friday.

“过去式”和“一般过去时”的区别

过去式是动词的一个形态：work -> worked, go -> went。
一般过去时是一个句子时态：用过去式表达过去发生并结束的事。I worked yesterday.

动词三态：原形、过去式、过去分词、现在分词

为什么要背三态

一般过去时用过去式；完成时用过去分词；进行时用现在分词。这比单独背“时态名字”更实用。

否定句与疑问句：英语靠助动词来搬运语法

否定句

be 后加 not：I am not ready. / The file is not ready.
普通动词借 do：I do not know. / She does not know.
过去借 did：They did not sign it. 注意 sign 用原形。
将来 will not：We will not delay it.

疑问句

be 提前：Are you ready? / Was he there?
普通动词借 do：Do you agree? / Does she agree?
过去借 did：Did they sign it? 注意 sign 用原形。
完成时 have/has 提前：Have you finished?
句子类型肯定否定疑问
BeShe is ready.She is not ready.Is she ready?
一般现在She works here.She does not work here.Does she work here?
一般过去She worked here.She did not work here.Did she work here?
将来She will work here.She will not work here.Will she work here?
完成She has worked here.She has not worked here.Has she worked here?

疑问词与从句：先问类型，再套语序

疑问词问什么例句注意点
What什么What is this document?问事物、内容、动作。
Who谁Who sent the file?who 作主语时，后面不需要 do。
Whom谁，宾格Whom did you call?正式；口语常用 who。
Whose谁的Whose laptop is this?后面常接名词。
Which哪一个/哪些Which option is better?有限范围内选择。
Where哪里Where is the file?地点。
When什么时候When will we start?时间点。
Why为什么Why did they reject it?原因。
How如何How do we solve this?方式、状态、程度。
How much / many多少How much time? / How many files?much 接不可数；many 接复数可数。

直接疑问句 vs. 间接疑问句

直接疑问句：Where is the file? be/助动词提前。
间接疑问句：I know where the file is. 后面恢复陈述语序。
常见句型：Could you tell me where the file is? 不是 where is the file。
